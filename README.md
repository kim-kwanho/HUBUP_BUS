# HUBUP Q&A

행사·등록과 관련해 **자주 묻는 질문**, **버스(이동) 일정 변경 요청** 등을 웹으로 안내·접수하는 **허브업(HUBUP) 전용 프론트**입니다. 메인 허브(등록·커뮤니티)와 연동해 쓰도록 설계되어 있으며, 운영 정책에 따라 일부 기능(예: 문의)은 켜거나 끌 수 있습니다.

## 기술 스택

- **Next.js** (React)
- **Supabase** (데이터)
- **NextAuth** (관리자 등 로그인, 설정 시)

## 로컬에서 실행하려면

1. **Node.js**와 **npm**이 필요합니다.
2. 저장소를 받은 뒤 프로젝트 루트에서:

```bash
npm install
npm run dev
```

3. 기본 개발 서버 포트는 **3001**입니다. (같은 머신에서 다른 앱과 포트를 나눠 쓰기 위한 설정입니다.)

4. Supabase·로그인 등 **환경 변수**는 팀에서 안내받은 값으로 `.env.local`에 설정합니다. **비밀 키가 들어 있는 파일은 버전 관리에 포함하지 마세요.** (`.gitignore`에 일반적으로 제외됩니다.)

운영·배포·DB 시드 등 **내부 절차**는 저장소에 포함하지 않으며, 담당자에게 별도로 전달합니다.

## 배포

GitHub 등에 올린 뒤, Vercel 등 **Next.js를 지원하는 호스팅**에 연결하면 됩니다. 프로덕션에서는 호스팅 콘솔에 환경 변수를 등록하고, OAuth를 쓰는 경우 **승인된 리디렉션 URI**에 배포 URL이 반영되어 있어야 합니다.

## 라이선스

별도 파일이 없으면 저장소 설정 또는 담당 부서 정책을 따릅니다.
